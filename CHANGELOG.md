# Change Log

All notable changes to the "django-cotton-highlighting" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

- Internal release workflow and metadata improvements for Marketplace publishing

## [0.0.4] - 2026-03-05

- Added command for generating Cotton component docstring templates
- Added go-to-definition support for Cotton components
- Added diagnostics for unknown components and attributes
- Added component naming style option (`snake_case` or `kebab-case`)

## [0.0.1] - 2026-03-05

- Initial release
- Basic syntax highlighting for Cotton templates
- Component autocomplete based on file discovery
- Documentation parsing from `{% comment %}` blocks
- Template variable highlighting
- Configurable templates path
